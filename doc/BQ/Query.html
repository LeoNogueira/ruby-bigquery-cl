execute(sql) click to toggle source

This method execute a query in a table. Call this method with a “query_string” like: “SELECT count(*) from [examplebucket/tables/ponylist] WHERE is_magic = true;” A hash is returned with the API response.

# File lib/ruby-bigquery-cl/query.rb, line 11
def execute(sql)
  # Add a ";" if it's missing
  sql += ";" if not sql.end_with?(';')

  # Use a remote procedure call (RPC) to query
  data = {:params => {:q => sql}, :method => 'bigquery.query'}

  # Send a query request
  result = BQ.request(:post, data.to_json)

  return result
end
